# Calorie Counter (Search)

A simple calorie counter search tool.

- Loads food data from `foods.json`
- Search by food description
- Shows up to 25 results in a scrollable table
- Clear button resets input + results

## Files
Put these files in the same folder:
- `index.html`
- `styles.css`
- `app.js`
- `foods.json`
